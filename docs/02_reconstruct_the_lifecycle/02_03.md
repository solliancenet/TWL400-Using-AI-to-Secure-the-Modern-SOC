# Task 03 - Mark detection-failure points and telemetry blind spots

## Introduction

You can now see the whole chain — but Zava's SOC couldn't, and reconstructing the lifecycle is only half the job. The other half is diagnosing *why* the existing stack missed it. The dominant failure mode here is **uncorrelated signal buried in noise**: across every layer a signal *existed* — including a **real Defender for Cloud — AI threat protection alert** for the AI attack — but **nothing correlated those signals into one cross-layer incident**, and each one sat among the ~25 decoys where a busy analyst would never connect them. Naming each gap precisely — which signal fired, and why nothing acted on it across layers — is what turns your reconstruction into the "why detection failed" evidence for the Proof Through Scenario in Exercise 04.

## Description

You will walk your five-stage lifecycle from Task 02.01 (corroborated in Task 02.02) and classify each stage's detection posture: **did a signal exist?** and **did anything correlate it across layers?** You will record, per stage, the detection-failure points (signal present, no cross-layer correlation), then call out the AI layer as the dominant gap — an alert that **fired but was never correlated and stayed buried in the noise**. You'll capture this in a table you fill in as you go.

## Success criteria

- For each of the five stages you have classified the detection posture and identified the gap — in this scenario, every stage is a **failure point** (a signal was present but uncorrelated across layers and buried in the ~25-decoy noise)
- **Each detection gap named** _(Validation: each detection gap and uncorrelated-signal failure point named)_
- You have explicitly named the **AI layer** (poisoned grounding → Refund Agent probing) as the dominant gap — a **real Defender for Cloud AI alert that fired but was never correlated** to the identity, infrastructure, and endpoint stages
- The completed gap table is saved for use in Exercise 03 (response) and Exercise 04 (Proof Through Scenario)
